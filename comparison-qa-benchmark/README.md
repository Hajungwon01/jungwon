# Comparison QA Benchmark (NAVER)

## Overview
LLM의 비교형(Comparison) 질문 성능을 체계적으로 평가하기 위해
데이터셋과 평가 파이프라인을 설계·구축한 프로젝트입니다.

## Problem
기존 QA 벤치마크는 단일 사실 회상에 치우쳐 있어,
LLM의 실제 비교 추론 능력을 정밀하게 평가하기 어렵다는 한계가 있습니다.

## Approach
- Comparison QA 문제 유형 정의
- 비교 대상 엔티티 및 속성 메타데이터 설계
- LLM 응답 생성 및 EM/F1 기반 평가 파이프라인 구축

## My Role
- 벤치마크 설계 및 데이터셋 구축
- 평가 파이프라인 구현
- 실험 결과 분석

## Notes
- 일부 데이터 및 설정은 공개 범위에 맞게 축약되어 있습니다.
