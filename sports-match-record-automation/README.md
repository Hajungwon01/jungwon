# Automated Volleyball Match Record Generation (Awarded)

## Overview
배구 경기 데이터를 기반으로 경기 기록지를 자동 생성하는 시스템을 구현한
졸업작품 프로젝트로, 해커톤에서 우수상을 수상하였습니다.

## Problem
기존 경기 기록지는 수작업 입력에 의존하여
시간 소모와 오류 발생 가능성이 높았습니다.

## Approach
- 경기 데이터 구조 정의
- 기록지 자동 생성 로직 구현
- 입력 데이터 기반 결과 포맷 자동화

## My Role
- 전체 시스템 설계
- 데이터 처리 로직 구현
- 결과 검증 및 개선
