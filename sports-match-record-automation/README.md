# Automated Volleyball Match Record Generation (Awarded)

> 배구 경기 데이터를 기반으로 경기 기록지를 자동 생성하는 시스템을 구현한  
> 졸업작품 프로젝트로, 해커톤에서 우수상을 수상하였습니다.

---

## Overview
본 프로젝트는 배구 경기 중 발생하는 다양한 이벤트 데이터를 입력으로 받아  
경기 기록지를 자동으로 생성하는 시스템을 구현한 졸업작품 프로젝트입니다.

기존의 경기 기록 방식은 수작업 입력에 크게 의존하고 있어  
시간 소모가 크고, 휴먼 에러 발생 가능성이 높다는 문제가 있었습니다.  
이를 해결하기 위해 데이터 처리부터 결과 생성까지의  
**End-to-End 자동화 파이프라인**을 설계·구현하였습니다.

---

## Problem
- 경기 기록을 사람이 직접 작성해야 하는 비효율적인 프로세스
- 기록 누락 및 입력 오류로 인한 데이터 신뢰성 저하
- 경기 종료 후 기록 정리에 소요되는 추가 시간

---

## Approach
문제 해결을 위해 다음과 같은 방식으로 시스템을 설계하였습니다.

1. **경기 데이터 구조 정의**
   - 득점, 서브, 블로킹 등 주요 이벤트 정형화
   - 기록지 생성에 필요한 최소 단위 데이터 설계

2. **데이터 처리 및 판단 로직 구현**
   - 이벤트 흐름에 따른 기록 항목 자동 계산
   - 규칙 기반 로직을 활용한 기록 정합성 보장

3. **기록지 자동 생성**
   - 입력 데이터를 기반으로 경기 기록지 포맷 자동 생성
   - 결과물의 가독성과 실제 사용 가능성 고려

---

## Result
- 경기 기록 작성 시간 대폭 단축
- 수작업 대비 기록 오류 감소
- 해커톤 우수상 수상으로 프로젝트 완성도 및 실효성 인정

---

## My Role
- 문제 정의 및 전체 시스템 설계
- 데이터 처리 및 기록 생성 로직 구현
- 결과 검증 및 개선

본 프로젝트는 기획부터 구현, 결과 검증까지 전 과정을 주도적으로 수행하였습니다.
