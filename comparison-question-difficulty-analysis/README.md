# Comparison Question Difficulty Analysis

## Overview
Comparison 질문이 LLM에게 왜 더 어려운지를 분석하기 위해,
정답 패턴과 오류 유형을 중심으로 난이도 요인을 분석한 개인 연구 프로젝트입니다.

## Problem
LLM은 비교형 질문에서 정답을 맞히더라도
추론 과정이 불안정하거나 일관되지 않은 경우가 빈번합니다.

## Approach
- Comparison 질문 정답/오답 패턴 분석
- Sub-question 단위 성능 비교
- 오류 유형별 분류 및 난이도 요인 도출

## My Role
- 실험 설계 및 데이터 분석
- 오류 패턴 정리 및 해석
- 분석 결과 정리
