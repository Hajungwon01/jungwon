# RAG + VLM Benchmark (ETRI)

## Overview
RAG 기반 LLM과 VLM의 성능을 평가하기 위해
텍스트–이미지 결합 태스크를 포함한 벤치마크를 설계한 프로젝트입니다.

## Problem
기존 벤치마크는 텍스트 중심 평가에 한정되어 있어,
멀티모달 환경에서의 LLM/VLM 성능을 충분히 평가하기 어렵습니다.

## Approach
- RAG 기반 QA 태스크 설계
- 텍스트–이미지 결합 입력 구조 정의
- 멀티모달 응답 평가 기준 설계

## My Role
- 벤치마크 태스크 설계
- 데이터 구조 정의
- 평가 기준 정리
